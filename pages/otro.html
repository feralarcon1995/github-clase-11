• Git init Inicializa git en mi proyecto, una SOLA VEZ, por proyecto.
  • Git status Ver el estado de los archivos.
  • Git add Guardar los archivos, usamos el "." para guardarlos todos al mismo tiempo.
  • Git rm --name-del-archivo Borra el archivo del stagin area.
  • Git commit -m "titulo" Entre comillas, va el titulo del commit bien descriptivo.
  • Git branch Lista todas las ramas existentes en el repositorio.
